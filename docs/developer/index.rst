Developer Guide
===============

This guide shall help you to understand how the project is organized and with this knowledge you should be able to contribute to the project in a short amount of time. The :ref:`Project Structure` guide gives you an overview how the project is structured in general and should be your starting point. Please also have a look at the **CONTRIBUTING.md** file in the main folder of the repository which explains more in detail how to contribute to this project.
